# desafio-go-full-cycle
Desafio do curso Full Cycle de criar uma imagem em Golang com menos de 2MB que imprima na tela a mensagem "Full Cycle Rocks!"

Para baixar a imagem:

docker pull diazdiego/golight:prd

Ao executar o docker run, aparece como resultado da imagem:
![image](https://github.com/diazdiego1/desafio-go-full-cycle/assets/140215970/3c1c364f-2888-4882-bf3c-c5c4174b187a)

Conforme so solicitado a imagem tem menos de 2MB, mais especificamente 1.85MB:
![image](https://github.com/diazdiego1/desafio-go-full-cycle/assets/140215970/a60b60fa-52e6-4745-b41a-c58f9d7fef76)
